# Basic Programming Concepts

Estos ejercicios son tomados del libro **Computer Science: An Interdisciplinary Approach** [[link]](https://introcs.cs.princeton.edu/java/home/). 

1. **Números aleatorios uniformes** Escriba un programa que imprima cinco números aleatorios uniformes entre 0 y 1, su valor promedio, y sus valores mínimo y máximo. Use `Math.random()`, `Math.min()`, y `Math.max()`.

2. **Ordenamiento de tres**. Escriba un programa que tome tres argumentos de línea de comandos enteros y los imprima en orden ascendente. Use `Math.min()` y `Math.max()`.

> [!important]
> Para estos dos ejercicios, use solo características de Java use solo caracteristicas basicas de java.
